# ⚡ Site Fortnite - JAOG1V1

Um site sobre **Fortnite Battle Royale** criado pelo jogador **JAOG1V1**, com design inspirado no estilo da Epic Games.

## 🎮 Sobre o Projeto

Site completo sobre Fortnite com informações sobre o jogo, modos de jogo, raridade das armas, galeria de imagens e formulário de contato funcional.

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura do site
- **CSS3** — Estilização e animações
- **JavaScript** — Interatividade (efeito de digitação, scroll animado, carrossel, lightbox)
- **EmailJS** — Envio de mensagens pelo formulário de contato
- **Google Fonts** — Fontes Inter e Bebas Neue

## 📋 Funcionalidades

- Design responsivo (funciona em PC e celular)
- Efeito de digitação automática no hero
- Animações ao rolar a página
- Carrossel de modos de jogo arrastável
- Barras de poder animadas nas armas
- Galeria com lightbox (clique para ampliar)
- Formulário de contato que envia e-mail de verdade
- Menu hambúrguer no mobile
- Botão voltar ao topo
- Link direto para download no site oficial da Epic Games
